## general info

```
:h tabpage.txt
```

## usage

Open a file in a new tab:
```
:tabnew <file>
```
or
```
:tabe <file>
```

Close the current tab:
```
:tabc
```

Go to the next tab:
```
:tabn
```
or `gt` or `<C-PgUp>`

Go to the previous tab:
```
:tabp
```
or `gT` or `<C-PgDn>`

> {count}gt       Go to tab page {count}.  The first tab page has number one.
