## ins-completion

```
:h ins-completion
```

#### tag completion

Complete keywords from `tags` file.

```
:h compl-tag
```

CTRL-X CTRL-]

#### omni completion

> Completion is done by a function that can be defined by the user with the
'omnifunc' option.\
This is to be used for filetype-specific completion.

```
:h compl-omni
```

CTRL-X CTRL-O

```
:h compl-omni-filetypes
```

See `:h ft-c-omni` for c specific completion.
