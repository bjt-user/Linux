## usage

#### open a website

```
:open github.com
```
or just hit `o` and it will type in `:open` for you.

#### navigation

`H` to go back to the previous site

#### copy url of current site to clipboard

`yy`

#### bookmark a website

Bookmark the current site:
```
:bookmark-add
```
or `M`

#### remove bookmark

remove the current website from bookmarks:
```
:bookmark-del
```

#### show bookmarks

```
:bookmark-list
```
or `Sq`
