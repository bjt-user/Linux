## usage

#### open a website

```
:open github.com
```
or just hit `o` and it will type in `:open` for you.

#### bookmark a website

Bookmark the current site:
```
:bookmark-add
```
or `M`

#### remove bookmark

remove the current website from bookmarks:
```
:bookmark-del
```

#### show bookmarks

```
:bookmark-list
```
or `Sq`
