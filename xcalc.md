## general info

```
man xcalc
```

#### installation

```
sudo pacman -S xcalc
```

#### CE/C in calculators

`CE` stands for "clear entry" or "cancel entry".

`C` means just "clear" or "cancel".

#### keyboard shortcuts

See "Accelerators" in manpage.

Space/Backspace/Del - Clear (Entry) \
q - quit \
Enter - Return / Evaluate / Equal Sign
