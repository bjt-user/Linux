## usage

#### show documentation for item under cursor

`<space>` + `k`

#### delete a character

It is not `x` as in `vim`, it is `d`.

#### delete a line

`xd`

Then you can past it with `p`.

#### visual mode

It is `v` instead of `V`.

#### go to start of the line

`gh`

## ex commands

#### write and exit buffer

`:wbc`
