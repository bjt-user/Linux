list directories only:
```
tree -d
```

only go 2 levels deep:
```
tree -d -L 2
```

#### charset ascii

```
$ tree --charset=ascii
.
|-- hello_world
|   |-- hello_world
|   `-- hello_world.rs
`-- list_dir_contents
    |-- main
    `-- main.rs
```

#### colors

`-C` will print colors

#### html output

`-X` will print html

#### json output

`-J` will print json
